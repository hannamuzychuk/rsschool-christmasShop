# 🎄 Christmas Shop

A fixed layout two-page website created as part of the RS School Front-End course. The project represents a festive-themed gift shop with clean layout, structured code, and interactive UI elements.

---

## 🚀 Live Demo

👉 [https://hannamuzychuk.github.io/rsschool-christmasShop/](https://hannamuzychuk.github.io/rsschool-christmasShop/)

---

## 🖥️ Pages

### 🏠 Home Page (`index.html`)

* Hero section
* About section
* Slider section
* Best Gifts section
* CTA (Subscribe + Countdown)
* Footer with contacts

### 🎁 Gifts Page (`gifts.html`)

* Hero section with tabs
* Gifts catalog (grid layout)
* Footer with contacts

---

## ✨ Features

* Semantic HTML5 structure
* Responsive centered layout (max-width: 1440px)
* Flexbox & CSS Grid
* Smooth scrolling navigation
* Interactive UI (hover effects, transitions)
* Clickable contact elements:

  * Phone link (`tel:`)
  * Email link (`mailto:`)
  * Google Maps (opens in new tab)
* Navigation between pages
* Active states for menu and tabs
* Favicon support
